# How to run

1. cd folder: books-management/
2. run: npm install
3. run: npm start
4. cd books-management/src/vanilla/
5. run: npm install

# Development
go to: books-management/src/vanilla/

authors app
```js
npm run authors-js-css-dev
```
books app
```js
npm run books-js-css-dev
```
